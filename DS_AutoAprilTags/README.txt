3. once the project is open in the unity editor you can build the projects by clicking file in the top left of the editor and then click "Build and Run"

2. instal the build tools https://aka.ms/vs/17/release/vs_BuildTools.exe
3. open the unity editor on the current project 
4. go to edit on the top left 
5. click prefrencses near the bottom of the list 
6. once the prefrence menu opens go to external tools and set external script editor to vs code
